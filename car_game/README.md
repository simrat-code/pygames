# 2D Car Game for Children

Welcome to the 2D Car Game project! This game is designed for children aged 6-9 years old, providing a fun and engaging way to navigate through an obstacle course.

## Project Structure

The project is organized as follows:

```
car-game
├── src
│   ├── main.py          # Entry point of the game
│   ├── game
│   │   ├── __init__.py  # Initializes the game module
│   │   ├── car.py       # Contains the Car class
│   │   ├── obstacle.py  # Contains the Obstacle class
│   │   └── course.py    # Contains the Course class
│   └── assets
│       ├── __init__.py  # Initializes the assets module
│       └── sprites.py    # Manages game sprites
├── requirements.txt      # Lists project dependencies
├── .gitignore            # Specifies files to ignore in version control
└── README.md             # Project documentation
```

## Installation

To get started with the game, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   cd car-game
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Running the Game

To run the game, execute the following command:
```
python src/main.py
```

## Game Instructions

- Use the arrow keys to control the car.
- Navigate through the obstacle course while avoiding collisions.
- Have fun and enjoy the game!
